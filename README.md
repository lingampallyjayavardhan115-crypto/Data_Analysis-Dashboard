# Personal Finance Dashboard 📊💰

An interactive and automated Excel-based dashboard designed to help users track, analyze, and manage their personal finances effectively. 

## 🚀 Features

This project utilizes advanced Microsoft Excel features to provide a seamless financial tracking experience:
* **Interactive Visualizations:** Includes multiple dynamic charts to visualize cash flow, expense breakdowns, and savings trends over time.
* **Dynamic Data Filtering:** Implements Excel Slicers for seamless, one-click filtering of dashboard metrics (e.g., filter by month, year, or expense category).
* **Automated Summaries:** Powered by interconnected Pivot Tables that automatically aggregate raw transaction data into meaningful insights.
